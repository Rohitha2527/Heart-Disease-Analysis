❤️ **Heart Disease Data Visualization Dashboard**
📌 **Project Overview**

This project presents a story-driven data visualization dashboard built on a heart disease dataset.

The focus of the project is on visual exploration and insight communication, transforming raw medical data into meaningful graphical representations that highlight trends, comparisons, and patterns.

In addition to the dashboard, the application also includes a Heart Disease Risk Prediction feature that estimates the risk level based on basic health parameters such as age, gender, diabetes status, cholesterol, blood pressure, and heart rate.

The dashboard is developed using Flask and deployed online using Render.

🎯 **Project Goal**

- Visualize cardiovascular health data effectively

- Create a structured story flow using charts

- Highlight key patterns through graphical comparison

- Provide a simple risk prediction tool based on health parameters

- Deploy the visualization dashboard for online access

🛠 **Technologies Used**

- Python

- Pandas

- Matplotlib

- Seaborn

- Flask

- HTML / CSS

- Gunicorn

- Render

🧠 **Risk Prediction Feature**

The application includes a Heart Disease Risk Prediction tool where users can enter health parameters such as:

- Age

- Gender

- Diabetes status

- Cholesterol level

- Blood pressure

- Maximum heart rate

Based on these inputs, the system estimates the risk level as:

- **Low Risk**

- **Moderate Risk**

- **High Risk**

📊 **Dashboard Highlights**

The dashboard includes visual analysis of:

- Age distribution of patients

- Cholesterol level comparison

- Resting blood pressure patterns

- Maximum heart rate variations

- Target variable distribution

Visual comparison between heart disease affected and non-affected groups

Each visualization contributes to a structured analytical narrative.

📖 **Story-Based Dashboard Flow**

The dashboard is designed as a visual analytical journey, guiding the viewer step-by-step:

1. 📊 Demographic overview of patients

2. 📈 Health parameter distribution analysis

3. 🔍 Comparative visualization

4. ❤️ Target group comparison

5. 📌 Summary of visual insights

This structured storytelling approach enhances clarity and data interpretation.

About the Project

This project is designed to analyze heart disease data and identify significant trends and patterns related to heart health. Through the use of data visualization and analytical techniques, the system enables a deeper understanding of factors that contribute to heart disease.

The application integrates interactive Tableau dashboards and stories within a Flask-based web framework, offering an intuitive and reliable platform for exploring healthcare data and supporting informed decision-making.
